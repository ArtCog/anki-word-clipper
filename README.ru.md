# Anki Word Clipper

[English](README.md) · **Русский**

Браузерное расширение (Chrome + Firefox): выделяешь слово или фразу на любой странице — и в один-два клика оно улетает в Anki **вместе с предложением-контекстом и ссылкой на источник**. Всё работает локально через [AnkiConnect](https://ankiweb.net/shared/info/2055492159), никакие данные наружу не отправляются.

## Установка

### 1. Anki

1. Установи и запусти Anki Desktop.
2. Установи аддон **AnkiConnect**: `Tools → Add-ons → Get Add-ons…` → код `2055492159` → перезапусти Anki.
3. Anki должен быть запущен, когда пользуешься расширением.

### 2. Расширение — Chrome / Edge / Brave

1. `chrome://extensions` → включи **Developer mode**.
2. Скачай ZIP из [Releases](https://github.com/ArtCog/anki-word-clipper/releases) и распакуй в постоянную папку (или клонируй репозиторий).
3. **Load unpacked** → выбери эту папку.
3. Предупреждения про `background.scripts` и `browser_specific_settings` — норма (это ключи для Firefox), Chrome их просто игнорирует.

### 3. Расширение — Firefox

1. `about:debugging#/runtime/this-firefox` → **Load Temporary Add-on…** → выбери `manifest.json`.
2. В `about:addons` → Anki Word Clipper → **Permissions** → включи «Access your data for all websites» (в Firefox MV3 доступ к страницам выдаётся вручную).

### 4. Первое подключение

При первом обращении Anki покажет диалог «An application is attempting to access Anki» — нажми **Yes**. Расширение автоматически создаст тип заметки **Word Clipper** (поля: Word, Translation, Context, Source, AddReverse; вторая карточка «перевод → слово» создаётся только если стоит галочка «обратная карточка»).

## Использование

- **Выдели текст** → рядом появится кнопка **«+ Anki»** → клик → мини-форма: слово и контекст уже подставлены, курсор в поле перевода → `Enter`.
- **Автоперевод:** поле «Перевод» заполняется само. Движок выбирается в попапе (секция «Перевод»): **Google** (из коробки, без настройки), **DeepL** (качественнее, бесплатный ключ на [deepl.com/pro-api](https://www.deepl.com/pro-api)) или **ИИ**. Если выбранный движок не отвечает — подстраховывает Google. Правишь слово в форме — перевод обновляется сам; введённое вручную не затирается.
- **ИИ-движок:** переводит слово по контексту предложения и превращает его в словарную форму (`Häusern` → `das Haus, die Häuser`) с краткой грамм. пометкой. Провайдер любой OpenAI-совместимый: Google Gemini (бесплатный ключ на [aistudio.google.com](https://aistudio.google.com/apikey)), OpenRouter, OpenAI, локальная Ollama (без ключа). Кнопка «Проверить ИИ» в попапе покажет пробный ответ.
- **Типы карточек:** обычная, двусторонняя (+ обратная «перевод → слово»), cloze (пропуск слова в твоём предложении). Выбор — в форме, дефолт — в попапе.
- **Озвучка (Anki TTS):** в попапе выбери язык — карточки будут произносить слово голосом системы, без внешних сервисов.
- **Перевод можно оставить пустым** — рабочий сценарий «захвати сейчас, переведи вечером в Anki».
- **Форму можно перетаскивать** за шапку «+ Anki», если она закрывает текст.
- **Мгновенный режим** (включается в попапе расширения): клик по кнопке добавляет карточку сразу, без формы. `Shift+клик` всегда делает наоборот (форма ↔ мгновенно).
- **ПКМ по выделению** → «Добавить в Anki…» (форма) или «Добавить в Anki мгновенно».
- **Hotkey** `Alt+A` — открыть форму для текущего выделения.
- **Дубликаты:** если слово уже есть, расширение скажет об этом и предложит «Добавить всё равно».
- **Обратная карточка:** галочка в форме (запоминается); в мгновенном режиме — настройка в попапе.

## Известные ограничения

- Не работает на служебных страницах (`chrome://`, Chrome Web Store, addons.mozilla.org).
- **Google Docs** не поддерживается (текст рисуется на canvas — выделение не видно расширению).
- **Chrome PDF-viewer:** плавающей кнопки нет; работает только ПКМ → «Добавить мгновенно» (без контекста предложения). Результат — галочка/восклицательный знак на иконке расширения.
- Карточки появятся на телефоне только после синхронизации Anki ↔ AnkiWeb.

## Приватность

Карточки живут только в твоём Anki (`http://127.0.0.1:8765`, локальный AnkiConnect). Ни аналитики, ни облака. Единственное исключение — **автоперевод**: включённый (по умолчанию), он отправляет выделенное слово в Google Translate или DeepL. Выключи галочку «Автоперевод» в попапе — и расширение снова работает на 100% локально.

## Разработка

Без сборки: репозиторий = готовое расширение. Чистая логика вынесена в `context-extract.js` и `anki-client.js` и покрыта тестами:

```bash
node --test tests/context-extract.test.js tests/anki-client.test.js
```

Иконки: `bash scripts/make-icons.sh` (нужен ffmpeg; путь к шрифту — переменная `FONT`).
